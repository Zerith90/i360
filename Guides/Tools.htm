1.     Email Templates

To view or modify email templates, go to Tools à Email Templates. Select the type of email templates from the dropdown. The types are as follows Nomination Notification Email, Nomination Reminder Email, Participants Reminder Email, Participants Notification Email.

Nomination Notification Email template is used when sending notification emails to superiors to inform them to nominate raters to his/her subordinates. Nomination Reminder Email template is used when sending reminder emails to superiors to remind them to nominate raters to his/her subordinates. Participants Notification Email template is used when sending notification emails to raters to inform them to log in to the system to give rating input to their assigned targets. Participants Reminder Email template is used when sending notification emails to raters to remind them to log in to the system to give rating input to their assigned targets.

All the wordings in the default template can be modified, except for the wordings within the < > tag. Click on the Default Template button to revert to the original template that was provided by the system.

Click on the Save button to save any updates.

2.     Send Nomination Emails

To send nomination emails, go to Tools à Send Nomination Emails.

Administrator can send out email notifications or reminders to all Supervisors or specific Supervisor to complete the nomination process. Should the server failed to send out email(s) due to network unavailability or any other reasons, a copy of the email(s) will be stored in the server and will be resent at a later time.

Figure 2. Send Nomination Emails

3.     Send Participant Email

To send participant email, go to Tools à Send Participant Email

Administrator can send out email notifications or reminders to all Raters or specific Rater to complete the providing feedback process. Should the server failed to send out emails due to network unavailability or any other reasons, a copy of the email(s) will be stored in the server and will be resent at a later time.

Figure 3. Send Participants Emails

4.                 View Sent Failed Emails

To view sent failed emails, go to Tools à View Sent Failed Emails

This lists all the emails that were failed to be sent out. Administrator would be able to either delete them or resend them.

Figure 4. Send Failed Emails

5.                 Import and Export

To view import and export, go to Tools à Import and Export

This provides Administrator with the feature to import and export data like user details, system libraries and Target & Rater’s assignment into the database efficiently. To retrieve the import template, simply do an Export of the category that the administrators wish and replace the data with the data that he/she wants to import.
