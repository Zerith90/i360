The data entry staff has three main roles:

·         Generate data collection questionnaire

·         Input collected data

·         Generate Outputs and reports

Once assigned with an ID and password, the DES can then access the system with this ID. (See Login section under Administrator Guide).

1.     Generate Data Collection Questionnaire

Under normal circumstances, the administrator would get the raters to enter their responses by accessing the system directly. However, if there is a requirement, the rating responses can be collected via questionnaire generated from the system. Data collected is then entered into the system by the data entry staff or the administrator. MS Excel document formatted questionnaire can be generated from the system.

Click on Reports icon on the left margin and then select Questionnaire.

Figure 1. Questionnaire Report

Select the survey, target and rater to generate the questionnaire. The questionnaire is a MS Excel document and administrator can modify the look and feel, add introduction, and cover page or any other text to it easily. It can be printed out to be faxed or emailed as an attachment for rating.

2.     Data Entry

Once data is collected, the DES can enter the data by accessing the questionnaire by clicking on Rater’s Data Entry icon on the left margin.

Figure 2. Rater’s Data Entry

Select the appropriate survey, group, target and rater and click Open Questionnaire.

Data can also be input by importing the ratings over with the click of a button if the rating was filled out via MS Excel sheet.

All the above function is similarly available for the administrator.
